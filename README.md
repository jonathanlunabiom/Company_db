## Description

This is an example of the behavior of a company, we simulate the interaction with data from a database. Where you can storage, manage and view all this information.

## Usage

1. Download all the packages needed for this app (inquirer and mysql).
2. Set your own mysql password on the createConnection.
3. Run the schema.sql and the seeds.sql in case you want an example to work on.
4. Do node server to start the server.
5. You will be prompted with a list of options to work on.
6. Select either view all employees or uptade one.
